Treebank Semantics Parsed Corpus links

The Treebank Semantics Parsed Corpus (TSPC) is a corpus of English with hand worked analysis for approaching half-a-million words.

You can find out about the TSPC by visiting the corpus Homepage.

The annotation scheme is documented in a Parsing Guide.

The following links take you to tables of words extracted from the TSPC.

  • Verb overview
  • Adverb overview
  • P-CONN overview
  • P-ROLE overview

There is also a version of the corpus, called Mor TSPC, which supplements the tree analysis with extra morphological analysis generated with the MOR program of the TalkBank project.
