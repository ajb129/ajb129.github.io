The Passive (2)


20.1    The passive with Be Going To and Modals

You can use pattern (20.1) to find passive clauses with be going to.

(20.1)
/LGS/ > (/^IP-PPL-CAT/ > (/^IP-INF/ $ (/^V/ < /^go/)))

For example, (20.2) is found with (20.1).

(20.2)

    You can use pattern (20.3) to find passive clauses with modals.

(20.3)
/LGS/ > (/^IP-PPL-CAT/ $ /^MD/)

For example, (20.4) is found with (20.3).

(20.4)

20.2    Get passives

You can use pattern (20.5) to find passive clauses with GET.

(20.5)
/LGS/ > (/^IP-PPL-CAT/ $ (/^V/ < /^(get|got)/))

For example, (20.6) is found with (20.5).

(20.6)

20.3    Passive gerunds and infinitives

You can use pattern (20.7) to find passive gerunds.

(20.7)
/LGS/ > (/^IP-PPL-CAT/ > (/^IP-PPL/ [ == /PPL$/ | == /-SBJ/ | == /-OB/ ]))

For example, (20.8) is found with (20.7).

(20.8)

    You can use pattern (20.9) to find passive infinitive clauses. What is found with (20.9) will include all of what is found with (20.1).

(20.9)
/LGS/ > (/^IP-PPL-CAT/ > /^IP-INF/)

For example, (20.10) is found with (20.9).
