Unit 24
Real Conditionals: Present and Future

You can use pattern (24.1) to find constituents that have a conditional function.

(24.1)
/CND/ < __

24.1   Present real conditionals

To form present real conditionals, you should use the simple present in the if clause and the main clause. Pattern (24.2) will find conditionals with this use of tense.

(24.2)
/CND/
  < (/^IP/ < (/^(BEP|HVP|DOP|VBP)/ !$ /CAT/))
  $ (/^(BEP|HVP|DOP|VBP)/ !$ /CAT/)

For example, (24.3) is found with (24.2).

(24.3)

But if you ask about their education , they pin you to the wall .


        [23_a_ted_talk_11@4,7,11,27]

You can use when or whenever in place of if without changing the conditional meaning. Pattern (24.4) will find examples with when or whenever and the same use of tense found with (24.2).

(24.4)
/SCON/
  < (/^P-CONN/ < /^when/)
  < (/^IP/ < (/^(BEP|HVP|DOP|VBP)/ !$ /CAT/))
  $ (/^(BEP|HVP|DOP|VBP)/ !$ /CAT/)

For example, (24.5) is found with (24.4).

(24.5)

you ’re very unpleasant when you ’re like that

24.2   Future real conditionals

To form future real conditionals, you should use the simple present in the if clause and a future verb form in the main clause. Pattern (24.6) will find conditionals with this use of tense.

(24.6)
/CND/ < (/^IP/ < (/^(BEP|HVP|DOP|VBP)/ !$ /CAT/)) $ (/^MD\b/ < /^w/)

For example, (24.7) is found with (24.6).

(24.7)

If they are not ellipsoids , the conclusions will be a reasonable approximation .


        [56_susanne_j03@2,5,9,23,24]

24.3   Real conditionals with modals, modal-like expressions, and imperatives

Pattern (24.8) finds present and future real conditionals with the modals in the main clause.

(24.8)
/CND/ < (/^IP/ < (/^(BEP|HVP|DOP|VBP)/ !$ /CAT/)) $ /^MD\b/

For example, (24.9) is found with (24.6).

(24.9)

If this description sounds familiar , you could enter the 1993 Sainsbury ’s Wine Taste Challenge .


        [83_lucy_bnc_b18@2,5,11,21]

(24.10)
/CND/ < (/^IP/ < (/^(BEP|HVP|DOP|VBP)/ !$ /CAT/)) > /IMP/

For example, (24.11) is found with (24.10).
