Unit 15
That Clauses


15.1   Finding that clauses

That clauses can be found with pattern (15.1). This searches for nodes with a label that starts with CP-THT. There is also a check that the found node has clause content, since it must dominate at least two further node levels.

(15.1)
/^CP-THT\b/ < (__ < __)

That clauses may occur with an overt complementizer, which is essentially always the word that. Instances of complementizers are found with pattern (15.2).

(15.2)
/^C\b/ < __

Both patterns (15.1) and (15.2) will find (15.3).

(15.3)
The high quality of the seam on Milos ensured that the area remained popular with early travelers .

That clauses may also occur without an overt complementizer. Such cases are found with pattern (15.4).

(15.4)
/^CP-THT/ < (__ < __) !<< (/^C\b/ < __)

15.2   Verbs that select object that clauses

Pattern (15.5) will find verbs that select a that-clause with object function (CP-THT-OB1) as the only selected (non-subject) complement.

(15.5)
__ > /~Tf\b/

Pattern (15.6) will find verbs that select a noun phrase with indirect object function (NP-OB2) and a that-clause with direct object function (CP-THT-OB1).

(15.6)
__ > /~Dn\.f\b/

Pattern (15.7) will find verbs that select a preposition phrase with indirect object function (PP-OB2) and a that-clause with direct object function (NP-OB1).

(15.7)
__ > /~Dpr\.f\b/

15.3   That clauses that are objects for present tense verbs of a main clause

Pattern (15.8) will match a that clause with object function, provided it occurs as the selected complement of a present tense verb in the matrix clause of a statement sentence.

(15.8)
/CP-THT-OB1/
  > (__=mid
       << (/^VBP\b/=verb !>> (/^(IP|FS)\b/ >> =mid))
       [ == /^IP-MAT\b/ | !== /^(IP|FS)\b/ >> (/^IP-MAT\b/ !<< (/^IP\b/ << =mid)) ]
    )
  • CLAIM: A present form in the that clause expresses a fact or general truth. Look at examples found with pattern (15.9) to check this claim.
(15.9)
/CP-THT-OB1/
  > (__=mid
       << (/^VBP\b/=verb !>> (/^(IP|FS)\b/ >> =mid))
       [ == /^IP-MAT\b/ | !== /^(IP|FS)\b/ >> (/^IP-MAT\b/ !<< (/^IP\b/ << =mid)) ]
    )
  < (/^IP\b/=i
       << (/^(HVP|BEP|DOP|VBP)\b/
             !$ (/^IP-PPL-CAT/ !< /LGS/)
             !>> (/^IP\b/ >> =i)
          )
    )

15.5   That clauses after adjectives

Pattern (15.18) finds sentences where a that clause is selected by an adjective.

(15.18)
__ > (/^(ADJ|ADJR|ADJS)\b/ $ CP-THT)

For example, (15.19) is found with pattern (15.18).

(15.19)
Pamela was glad Jim was nowhere near .

15.6   That clauses after nouns

Pattern (15.20) finds sentences where a that clause is selected by a noun.

(15.20)
__ > (/^(N|NS)\b/ $ CP-THT)

For example, (15.21) is found with pattern (15.20).

(15.21)
Hey_everyone_, this is n't a situation I ever imagined or intended when I posted a few days ago .

15.7   That clause extraposition

Note that it is possible for a that clause to immediately follow a noun or adjective and yet not have been (directly) selected by the noun or adjective. One prominent case of this involves that clause extrapositioning, e.g., search for:

(15.22)
/PRD2/ $ /PSBJ/ $ /CP-THT-NSBJ/

For example, (15.23) is found with pattern (15.22).

(15.23)
It is a sweeping generalisation that greed and selfishness motivate a modern capitalist society .
