Unit 22
Object relative clauses

You can use pattern (22.1) to find object relative clauses.

(22.1)
/OB/ [ < /^RPRO\b/ | < /^\*T\*/ ]

Pattern (22.1) finds object relative clauses by locating a constituent with object function that is either made up of a relative pronoun or a zero element.

  • A relative pronoun is a word tagged RPRO, e.g. which, who, whom, that.
  • A zero element is an element which is required by the clause, but has an absence that is construction licensed.

Such objects link the internal object for the relative clause to the overall noun phrase contribution of which the relative clause is a part.


22.1   Restrictive object relative clauses

Like subject relative clauses, object relative clauses can be either restrictive or non-restrictive. In written language, restrictive relative clauses are not immediately preceded by punctuation. Pattern (22.2) finds object relative clauses that are not immediately preceded by punctuation.

(22.2)
/^IP-REL/=rel
  << (/OB/ [ < /^RPRO\b/ | < /^\*T\*/ ] !>> (/^IP-REL/ >> =rel))
  !$, /^PU/
  $ __

For example, (22.3) is found with (22.2).

(22.3)
Every movement she made seemed unnecessarily noisy .

22.2   Non-restrictive object relative clauses

In written language, we can find non-restrictive relative clauses by ensuring that the found relative clause is immediately preceded by punctuation. Such a search is accomplished with pattern (22.4).

(22.4)
/^IP-REL/=rel << (/OB/ [ < /^RPRO\b/ | < /^\*T\*/ ] !>> (/^IP-REL/ >> =rel)) $, /^PU/

For example, (22.5) is found with (22.4).

(22.5)
elastic bands , which postmen discard when delivering letters

22.3   Object relative clauses with long distance displacements

(22.6)
/REL/ << (/OB/ < /^\*ICH\*/)

For example, (22.7) is found with pattern (22.6).

(22.7)
the beast that I thought I loved

22.4   Relativised content integrated with prepositions

Pattern (22.8) will find relative clauses where there is relative pronoun or zero element that is integrated into the content of the relative clause with a preposition. Note that neither the preposition nor the containing preposition phrase needs to be mentioned in the pattern. This is because a relative pronoun or zero element will need to occur with functional information and since the pattern says it falls under an NP that has no functional information itself it will have to be the case that the NP is inheriting functional information from a PP layer of structure.

(22.8)
/^NP\b/ [ < /^RPRO\b/ | < /^\*T\*/ ] > /^PP\b/

For example, (22.9) and (22.10) are found with pattern (22.8).

(22.9)
The method of assessment to which the Commons agreed
(22.10)
I always crack the mirrors I look in

It is also possible for relativised content to be integrated with prepositions while also involving displacement. Pattern (22.11) finds such examples.

(22.11)
/REL/ << (/^NP\/ < /^\*ICH\*/ > /^PP\b/)

For example, (22.12) is found with pattern (22.11).

(22.12)
a club book which we put our password and our motto in
